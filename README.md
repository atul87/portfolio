## 📌 Overview

This portfolio is not just a static website — it is a **production-ready full-stack application** with:

* Dynamic UI built using modern frontend technologies
* Backend API for handling contact form submissions
* Real-time email delivery system
* CI/CD pipeline with automatic deployments

---

## 🛠️ Tech Stack

### Frontend

* Next.js (App Router)

* Tailwind CSS

### Backend

* Next.js API Routes

## ⚙️ Environment Variables

Create a `.env.local` file in the root directory:

```env
EMAIL_ADDRESS=your_email@gmail.com
GMAIL_PASSKEY=your_app_password
```

> ⚠️ Use a Google App Password, not your actual Gmail password.

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/atul87/portfolio.git
cd portfolio
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run the development server

```bash
npm run dev
```

### 4. Open in browser

```text
http://localhost:3000
```
